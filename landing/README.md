# Much Better Authenticator Landing Page

React + Vite + Tailwind landing page for Much Better Authenticator.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, usually:

```text
http://localhost:5173
```

## Build

```bash
npm run build
npm run preview
```

No external icon package is used. Icons are inline SVGs inside `src/App.jsx`.
