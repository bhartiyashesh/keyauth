<threat_model>
- **T1: Clipboard persistence** -- TOTP codes should not linger in clipboard. The 30-second auto-clear timer starts on copy, not on code receipt. Uses `navigator.clipboard.writeText('')` to clear. If popup is closed before timer fires, the clear may not execute (best-effort, same pattern as 1Password/Bitwarden).
- **T2: Stale code display** -- Code must auto-dismiss when the TOTP 30-second period expires. The countdown timer is based on `codeReceivedAt` timestamp from session storage, not a React-local timer, so it survives popup close/reopen.
- **T3: Code displayed after unpair** -- If user unpairs while code is displayed, the state machine transitions to unpaired state, clearing the code view. The `chrome.storage.onChanged` listener in App.tsx handles this.
- **Mitigations applied:** Auto-dismiss on expiry, clipboard auto-clear, state derived from storage (not component state).
</threat_model>

    **Code display:**

    - Render in large monospace font: `font-family: 'SF Mono', 'Menlo', 'Consolas', monospace; font-size: 36px; font-weight: 600; letter-spacing: 4px`.

    **Circular countdown ring (SVG):**
    - Calculate `secondsRemaining` from TOTP 30-second period: `30 - (Math.floor(Date.now() / 1000) % 30)`.
    - SVG circle (size 48px): outer gray track circle, inner progress circle.
    - Progress: `strokeDashoffset = circumference * (1 - secondsRemaining / 30)`.
    - Color: green (#22c55e) when >5s remaining, red (#ef4444) when <=5s.
    - Center text: `{secondsRemaining}s` inside the ring.
    - `useEffect` with 1-second `setInterval` to update countdown.
    - Rotate SVG -90deg so progress starts from top.

    **Auto-dismiss on expiry:**
    - When `secondsRemaining` reaches 0, call `onDismiss()`.
    - `onDismiss` in App.tsx clears `lastCode` from session storage and sets `connectionState` back to `'connected'`.

    **Copy button:**

      1. `await navigator.clipboard.writeText(code)` (the raw 6-digit string, no spaces).
      2. Show "Copied!" toast: set local state `copied = true`, render small green text "Copied!" below button.
      3. Auto-hide toast after 2 seconds.
      4. Start 30-second clipboard clear timer: `setTimeout(() => navigator.clipboard.writeText('').catch(() => {}), 30_000)`.
    - Per manifest permissions, `clipboardWrite` is already declared.

    **Layout (top to bottom):**
